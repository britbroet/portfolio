ViceGrip

View demo video

WHAT

ViceGrip is an app that allows users to track their bad habits, the money they're saving/spending on them, and to set financial goals to work toward with the money they save from cutting back.

WHO

This was a team project with four collaborators including myself.

HOW

ViceGrip was built with Ruby on Rails and PostgreSQL.

WHERE

Live App
GitHub Repo
