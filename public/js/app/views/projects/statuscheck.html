StatusCheck

View demo video

WHAT

StatusCheck is an app that allows users to create and share task lists to keep others easily informed as to their progress.

WHO

Just me.

HOW

StatusCheck was built with JavaScript, jQuery, NodeJS, PostgreSQL, and uses the Twilio API to send text updates.
